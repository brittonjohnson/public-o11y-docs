**************************************************
Set your on-call notification preferences
**************************************************

.. meta::
   :description: Set your on-call notification preferences for Splunk Observability Cloud for Mobile. 

If you are receiving on-call notifications via mobile push or SMS and want to change these notification methods, you can set your on-call notification preferences in Splunk Observability Cloud. See :ref:`ii-notification-preferences`. You can customize all other notification settings, including sound, volume, vibrate, etc., using your device OS.
